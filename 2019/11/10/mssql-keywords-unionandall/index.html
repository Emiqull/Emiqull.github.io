Q&A / UNION 和 UNION ALL

文章目录

语法格式
语法区别
语法应用

语法格式

  • UNION：SELECT … UNION SELECT … ;
  • UNION ALL：SELECT … UNION ALL SELECT … ;

语法区别

  • UNION是对多个SELECT查询结果集的列数据进行合并，并对合并后的结果集去除重复行且默认按照第一条SELECT语句的第一列的值进行升序排列

    • 若第一条SELECT语句查询的是，则默认先转换成所有列，再按照所有列的第一列进行排序
    • 对UNION连接后的最终结果集的排序，只能在UNION连接的最后一个SELECT语句末尾添加

  • UNION ALL 仅仅对多个SELECT查询结果集的列数据进行合并

    • 对UNION ALL连接后的最终结果集的排序，只能在UNION ALL连接的最后一个SELECT语句末尾添加

语法应用

  • 为了更好的利用系统资源，如果多个SELECT语句的结果，在确定不需要去除重复行、对连接后的结果集不需要排序后，建议使用UNION ALL连接
