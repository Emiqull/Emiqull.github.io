值类型比较

运算符：相等运算符 ==
定义：如果操作数相等，等于运算符 == 返回 true，否则返回 false
格式：a == b
返回值注解：如果置值类型的值相等，则其操作数相等
详细内容及使用示例：https://docs.microsoft.com/zh-cn/dotnet/csharp/language-reference/operators/equality-operators#value-types-equality

引用类型比较

方法：Object.ReferenceEquals
定义：确定指定的 Object 实例是否是相同的实例
格式：public static bool ReferenceEquals (object objA, object objB);
返回值注解：如果 objA 是与 objB 相同的实例，或两者均为 null，则为 true；否则为 false
详细内容及使用示例：https://docs.microsoft.com/zh-cn/dotnet/csharp/language-reference/operators/equality-operators#reference-types-equality

字符串类型比较

方法：String.Compare
定义：比较两个指定的String对象，并返回一个指示二者在排序顺序中的相对位置的整数
格式：public static int Compare (string strA, string strB)
返回值注解：Compare 方法的所有重载返回一个 32 位有符号整数，指示两个比较数之间的词法关系:

  • 小于零 在排序顺序中，第一个子字符串在第二个子字符串之前
  • 零 子字符串在排序顺序中出现的位置相同或 length 为零
  • 大于零 第一个子字符串在排序顺序中后跟第二个子字符串
    详细内容及使用示例：https://docs.microsoft.com/zh-cn/dotnet/api/system.string.compare?view=netframework-4.8

方法：String.CompareOrdinal
定义：通过计算每个字符串中相应 String 对象的数值来比较两个 Char 对象
格式：public static int CompareOrdinal (string strA, string strB);
返回值注解：一个整数，指示两个比较字之间的词法关系：

  • 小于零 strA 小于 strB
  • 零 strA 与 strB 相等
  • 大于零 strA 大于 strB
    详细内容及使用示例：https://docs.microsoft.com/zh-cn/dotnet/api/system.string.compareordinal?view=netframework-4.8

方法：String.CompareTo
定义：将此实例与指定对象或 String 进行比较，并返回一个整数，该整数指示此实例在排序顺序中是位于指定对象或 String 之前、之后还是与其出现在同一位置
格式：public int CompareTo (string strB);
返回值注解：一个 32 位带符号整数，该整数指示此实例在排序顺序中是位于 strB 参数之前、之后还是与其出现在同一位置：

  • 小于零 此实例位于 strB 之前
  • 零 此实例在排序顺序中的位置与 strB 相同
  • 大于零 此实例位于 strB 之后。 或 strB 为 null
    详细内容及使用示例：https://docs.microsoft.com/zh-cn/dotnet/api/system.string.compareto?view=netframework-4.8

方法：String.Equals
定义：确定两个 String 对象是否具有相同的值
格式：public bool Equals (string value);
返回值注解：如果 true 参数的值与此实例的值相同，则为 value；否则为 false。 如果 value 为 null，则此方法返回 false
详细内容及使用示例：https://docs.microsoft.com/zh-cn/dotnet/api/system.string.equals?view=netframework-4.8

Sequence比较

方法：Queryable.SequenceEqual
定义：确定两个序列是否相等
格式：public static bool SequenceEqual (this System.Linq.IQueryable source1, System.Collections.Generic.IEnumerable source2);
返回值注解：如果两个源序列的长度相等并且它们的对应元素也相等，则为 true；否则为 false
注意：该方法的第一个参数类需要实现IQueryable接口，如果参数实现了IEnumeralbe，使用Queryable.AsQueryable方法将 IEnumerable 转换为 IQueryable
详细内容及使用示例：https://docs.microsoft.com/zh-cn/dotnet/api/system.linq.queryable.sequenceequal?view=netframework-4.8

HashSet比较

方法：HashSet.SetEquals
定义：确定是否 HashSet 对象和指定集合包含相同的元素
格式：public bool SetEquals (System.Collections.Generic.IEnumerable other);
返回值注解：如果 HashSet 对象等于 other，则为 true；否则为 false
详细内容及使用示例：https://docs.microsoft.com/zh-cn/dotnet/api/system.collections.generic.hashset-1.setequals?view=netframework-4.8

DateTime比较

方法：DateTime.Compare
定义：对两个DateTime的实例进行比较，并返回一个指示第一个实力是早于、等于还是晚于第二个实例的整数
格式：public static int Compare (DateTime t1, DateTime t2);
返回值注解：有符号数字，指示 t1 和 t2 的相对值

  • 小于零 t1 早于 t2
  • 等于零 t1 与 t2 相同
  • 大于零 t1 晚于 t2
    详细内容及使用示例：https://docs.microsoft.com/zh-cn/dotnet/api/system.datetime.compare?view=netframework-4.8

方法：DateTime.Compareto
定义：将此实例的值与指定的DateTime值比较，并指示此实例是早于、等于还是晚于指定的DateTime值
格式：public int CompareTo (DateTime value);
返回值注解：有符号数字，指示此实例和value参数的相对值

  • 小于零 此实例早于 value
  • 零 此实例与 value 相同
  • 大于零 此实例晚于 value
    详细内容及使用示例：https://docs.microsoft.com/zh-cn/dotnet/api/system.datetime.compareto?view=netframework-4.8

方法：DateTime.Equals
定义：返回一个值，该值指示两个DateTime对象，或者一个DateTime实例和另一个对象或DateTime是否具有相同的值
格式：public bool Equals (DateTime value)
返回值注解：如果value参数与此实例的值相等，则为true；否则为false
详细内容及使用示例：https://docs.microsoft.com/zh-cn/dotnet/api/system.datetime.equals?view=netframework-4.8

Uri比较

方法：Uri.Compare
定义：使用指定的比较规则比较两个URI的指定部分
格式：public static int Compare(Uri uri1, Uri uri2, UriCompents partsToCompare, UriFormat compareFormat, StringComparison comparisonType);
返回值注解：一个Int32值，指示进行比较的URI部分之间的此法关系

  • 小于零 uri1 小于 uri2
  • 零 uri1 等于 uri2
  • 大于零 uri1 大于 uri2
    详细内容及使用示例：https://docs.microsoft.com/zh-cn/dotnet/api/system.uri.compare?view=netframework-4.8

方法：Uri.Equals
定义：比较两个Uri实例是否相等
格式：public override bool Equals(object compared);
返回值注解：一个Boolean值。如果两个实例表示相同的URI，则该值为true；否则为false
详细内容及使用示例：https://docs.microsoft.com/zh-cn/dotnet/api/system.uri.equals?view=netframework-4.8

运算符：相等运算符==
定义：确定两个Uri实例是否具有相同的值
格式：public static bool operator == (Uri uri1, Uri uri2);
返回值注解：一个Boolean值。如果这两个Uri实例是等效的，则该值为true；否则为false
详细内容及使用示例：https://docs.microsoft.com/zh-cn/dotnet/api/system.uri.op_equality?view=netframework-4.8

Guid比较

方法：Guid.ComapreTo
定义：将此实例与指定的对象或Guid进行比较，并返回二者相对值的指示
格式：public int CompareTo (Guid value);
返回值注解：一个带符号数字，指示此实例和value的相对值

  • 负整数 此实例小于 value
  • 零 此实例等于 value
  • 正整数 此实例大于 value
    详细内容及使用示例：https://docs.microsoft.com/zh-cn/dotnet/api/system.guid.compareto?view=netframework-4.8

方法：Guid.Equals
定义：返回一个值，该值指示Guid的两个实例是否表示同一个值
格式：public bool Equals (Guid g);
返回值注解：如果g与该实例相等，则为true；否则为false
详细内容及使用示例：https://docs.microsoft.com/zh-cn/dotnet/api/system.guid.equals?view=netframework-4.8

Type比较

方法：Type.Equals
定义：确定当前Type的基础系统类型是否与指定Object或Type的基础系统类型相同
格式：public virtual bool Equals(Type o);
返回值注解：如果o的基础系统类型与当前 Type 的基础系统类型相同，则为 true；否则为 false
详细内容及使用示例：https://docs.microsoft.com/zh-cn/dotnet/api/system.type.equals?view=netframework-4.8

IntPtr比较

方法：IntPtr.Equals
定义：返回一个值，该值指示此实例是否等于指定的对象
格式：public override bool Equals (object obj);
返回值注解：如果 obj 是 IntPtr 的实例并且等于此实例的值，则为 true；否则为 false
详细内容及使用示例：https://docs.microsoft.com/zh-cn/dotnet/api/system.intptr.equals?view=netframework-4.8
