语法概念

  • 通过OFFSET FETCH子句，可以从结果集中仅提取某个时间范围或某一页的结果记录
  • OFFSET FETCH子句只能与ORDER BY 子句一起使用

语法格式

语法参数

OFFSET { integer_constant | offset_row_count_expression } { ROW | ROWS }

  • 指定在查询表达式开始返回结果记录之前，将要跳过的行数
  • OFFSET子句的参数可以是大于或等于零的整数或表达式
  • ROW和ROWS可以互换使用

FETCH { FIRST|NEXT } <行计数表达式> { ROW|ROWS } ONLY

  • 指定在经过OFFSET子句处理后，将要返回的结果记录数
  • FETCH子句的参数可以是大于或等于1的整数或表达式
  • ROW和ROWS可以互换使用
  • FIRST和NEXT可以互换使用

语法限制

  • OFFSET FETCH子句必须与ORDER BY 子句一起使用
  • FETCH关键字必须与OFFSET子句一起使用
  • TOP关键字不能在同一个查询表达式中与OFFSET FETCH子句一起使用
  • OFFSET 和 FETCH的行计数表达式可以是会返回整数值的任何算术、常量或表达式，但不支持标量子查询

注： 标量子查询就是在SELECT语句中出现的子查询
