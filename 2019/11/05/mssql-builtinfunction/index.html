字符串函数

CHARINDEX(TargetCharacter, SourceCharacter[, indexNumber]);
定义：返回[从源字符串指定索引开始]搜索目标字符串在源字符串中第一次出现的索引，若目标字符串在源字符串中不存在，则返回0
注意：若目标字符串与源字符串的内容是或全是NULL，则返回NULL

LEFT(TargetCharacter, lengthNumber);
定义：返回目标字符串从左侧开始，长度为指定整数的字符串表达式

RIGHT(TargetCharacter, lengthNumber);
定义：返回目标字符串从右侧开始，长度为指定整数的字符串表达式

LEN(TargetCharacter);
定义：返回目标字符串的字符数，即字符串长度
注意：返回的字符数不包含字符串右边空格，包含制表符、换行符

LOWER(TargetCharacter);
定义：返回目标字符串的小写表达式

UPPER(TargetCharacter);
定义：返回目标字符串的大写表达式

LTRIM(TargetCharacter);
定义：返回删除目标字符串左侧空格的表达式

RTRIM(TargetCharacter);
定义：返回删除目标字符串右侧空格的表达式

PATINDEX(TargetCharacter, SourceCharacter);
定义：返回目标字符串在源字符串中第一次出现的索引，若目标字符串在源字符串中不存在，则返回0
注意： 与CHARINDEX函数不同的是PATINDEX函数的目标字符串支持使用通配符，如SELECT (‘%ab%’, ‘123ab456’); –返回4、SELECT (‘%ab_’, ‘123ab456’); –返回0

REVERSE(TargetCharacter);
定义：返回目标字符串反转后的表达式

SPACE(spaceNumber);
定义：返回指定整数个的空格
注意：SPACE函数可使用在字符串拼接中，如 SELECT ‘A’+SPACE(4)+’B’; –返回A B

STR(floatNumber[, lengthNumber[, decimalLengthNumber]]);
定义：返回[指定整个字符串长度的[指定小数位长度的]]数值的字符串表达式
注意：

  • 数值的字符串长度小于指定长度时，字符串表达式前面补足空格、截断小数位遵守四舍五入、截取整数位返回**
  • 整个字符串长度包括小数点、符号、数字以及空格；总长都默认值为10且不保留小数；小数位长度最大是16

STUFF(SourceCharacter, beginNumber, lengthNumber, TargetCharacter);
定义：返回源字符串从指定开始索引位置开始删除指定长度的字符，并在该索引位置插入目标字符串后的字符串表达式

SUBSTRING(TargetCharacter, beginNumber, lengthNumber);
定义：返回目标字符串从指定开始索引位置截取的指定长度的字符串表达式

REPLACE(SourceCharacter, replaceCharacter, TargetCharacter);
定义：返回目标字符串替换源字符串中存在的所有替换字符串后的字符串表达式

REPLICATE(TargetCharacter, intNumber);
定义：返回以指定数字次数重复目标字符串后的字符串表达式
注意：若指定次数为负数，则返回NULL值；若指定次数为0，则返回空值

REPLICATE(TargetCharacter, intNumber);
定义：返回目标字符串重复指定次数后的字符串表达式
注意：若指定重复次数为负数，则返回NULL；若指定重复次数为0，则返回空

时间日期函数

DATEADD(datePart, intNumber, TargetDateTime);
定义：返回在目标日期的指定日期部分加上指定整数值后的日期表达式
注意：detePart常用格式1，YEAR日期年、MONTH日期月、DAY日期天、HOUR日期时、MINUTE日期分、SECOND日期秒

DATEDIFF(datePart, BeginDateTime, EndDateTime);
定义：返回结束日期与开始日期的时间差值的指定日期部分的数值
注意：该函数是计算日期间的差值，在返回指定差值日期部分的指定部分

DATENAME(datePart, TargetDateTime);
定义：返回目标日期的指定日期部分的名称
注意：datePart常用格式2，WEEKDAY星期数、DAYOFYEAR年中天

GETDATE();
定义：返回当前系统时间

GETUTCDATE();
定义：返回当前UTC(世界标准时间)时间

数学函数

ABS(Number);
定义：返回指定数值的绝对值

PI();
定义：返回π值

CEILIING(Number);
定义：返回向上舍入指定数值的整数值

FLOOR(Number);
定义：返回向下舍入指定数值的整数值

ROUND(Number, decimalIndexNumber[, funcationWay]);
定义：返回指定数值[按舍入方式]舍入指定小数索引位置的后一位后的数值且小数索引位置后全变为0的数值表达式
注意：

  • 函数方式默认为0，舍入方式遵守四舍五入；若指定其他整数值，则舍入方式等同于截取
  • 小数索引位置大于0舍入小数，小数索引位置小于0舍入整数
  • 当小数索引位置为负数且数值等于指定数值的整数部分长度，若舍入方式为截取，则返回0.再拼接指定数值的小数位个数的0
  • 若舍入方式为默认，当指定数值整数最高位小于5，则返回0.再拼接指定数值的小数位个数的0
  • 若指定数值整数最高位大于5，则ERROR

POWER(Number, powerNumber);
定义：返回计算指定数值的指定次幂后的数值

SQRT(Number);
定义：返回计算指定数值的平方根后的数值

SIGN(Number);
定义：返回指定数值的正负，若为正数返回1；若为负数返回-1；若为0返回0

数据类型转换函数

CAST(TargetExpression AS dataType[(LengthNumber)]);
定义：返回目标表达式转换为[指定长度的]指定数据类型的表达式

CONVERT(dateType[(LengthNumber)], TargetExpression[, timeStyle]);
定义：返回[指定日期显示样式的]目标表达式转换为[指定长度的]指定数据类型的表达式
补充：常见日期格式的转换方法

  • Select CONVERT(varchar(100), GETDATE(), 24): 10:57:47
  • Select CONVERT(varchar(100), GETDATE(), 108): 10:57:49
  • Select CONVERT(varchar(100), GETDATE(), 12): 060516
  • Select CONVERT(varchar(100), GETDATE(), 23): 2006-05-16
  • Select CONVERT(varchar(12) , getdate(), 112 ) : 20040912
  • Select CONVERT(varchar(100), GETDATE(), 8): 10:57:46

排序函数

ROW_NUMBER()
定义：SELECT ROW_NUMBER() OVER (ORDER BY 列名 [ASC|DESC]) AS 序号列名, * FROM 表名
注意：ROW_NUMBER函数会按照排名的顺序生成递增的排名序号

RANK()
定义：SELECT RANK() OVER (ORDER BY 列名 [ASC|DESC]) AS 序号列名, * FROM 表名
注意：RANK函数会把OVER子句中相同的字段作为一组进行排名，且每组的排名序号相同

聚合函数

AVG(列名);
定义：返回指定列的值的平均值
注意：AVG函数会忽略NULL值

MIN(列名);
定义：返回指定列的值的最小值
注意：MIN函数会忽略NULL值

MAX(列名);
定义：返回指定列的值的最大值
注意：MAX函数会忽略NULL值

SUM(列名);
定义：返回指定列的值的和
注意：SUM函数会忽略NULL值

COUNT(列名[, *]);
定义：返回当前记录源的总项数
注意：COUNT函数不会忽略NULL值

Null值判断函数

ISNULL(TargetExpression, replaceValue);
定义：若目标表达式的值为NULL，则返回指定的替换值；若目标表达式的值不为NULL，则返回目标表达式

COALESCE([TargetExpression1,TargetExpression2, … , ]TargetExpressionN);
定义：返回目标表达式中第一不为NULL的值
注意：目标表达式中至少要有一个不为NULL的值，否则会ERROR

其他函数

IIF(LogicalExpression, TrueExpression, FalseExpression);
定义：若逻辑表达式的结果为TRUE，则返回真表达式；若逻辑表达式的结果为FALSE，则返回假表达式

IDENTITY(dataType[, firstValue, stepValue]);
定义：IDENTITY函数只用在INSERT INTO … SELECT … 语句中，将标识列插入到目标表中
