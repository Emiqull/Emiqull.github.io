Hexo 基础命令

本地浏览

hexo g && hexo s

部署与发布

hexo clean && hexo g && hexo d

新建博客

hexo new <title>

创建分类

hexo new page categories

给文章添加分类(修改.md文件)

创建标签

hexo new page tags
